Variable name Role Type Description
fridgeID Feature Categorical Unique MetaFridge identifier, used as primary key in database tables, and in time series a composite PK along with datetime stamp.
measuredDtm Feature Datetime Time that the sensor values were recorded, with the time referenced to a real-time clock managed by the Device. The native datatype is datetime2(0), which is kept to a full second.
createdDtm Feature Datetime The time that the data was inserted into the database server. The native datatype is datetime2(3), which is kept to a millisecond.
